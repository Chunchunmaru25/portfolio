# 🌐 My Portfolio

Welcome to my personal portfolio! This project showcases my skills, projects, and experience as a full-stack web developer. It features a modern, responsive design where visitors can explore my portfolio, technical skills, development tools, resume, FAQs, and contact information.

## ✨ Features

* Responsive design for desktop, tablet, and mobile
* Light and Dark mode
* Interactive project showcase
* Skills and technology stack section
* Resume viewer
* Contact form
* Frequently Asked Questions (FAQs)
* Smooth animations and modern UI
* SEO-friendly structure

## 🛠️ Built With

### Frontend

* **React.js** – Building interactive user interfaces
* **TypeScript** – Type-safe JavaScript
* **Tailwind CSS** – Utility-first CSS framework
* **Shadcn/UI** – Beautiful and reusable UI components
* **ReactBits** – Reusable React components
* **UiVerse** – Community-made UI elements
* **SVGRepo** – SVG icons and illustrations

### Backend

* **Node.js** – JavaScript runtime
* **Express.js** – REST API and server

### Database & Backend Services

* **Supabase** – Backend-as-a-Service
* **PostgreSQL** – Relational database

### Development Tools

* **Vite** – Fast development and build tool
* **NPM** – Package manager
